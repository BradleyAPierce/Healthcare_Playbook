How You Can Add Value

Become familiar with the value of these two solutions:

  1. Dispatcher Paragon Cloud Fax
  2. OpenText RightFax

Ask prospects if they see value in a solution that offers these value adds:

  • Integration of fax documents with the E H R system or with other applications
  • Tracking and reporting of faxes
  • Expanded access to fax for clinical employees
  • Expanded interoperability across the complete community of healthcare providers
  • Replacement of legacy fax machines with secure digital document delivery
  • Elimination of telephony hardware

More about Dispatcher Paragon Cloud Fax

  • Provides HIPAA–compliant fax
  • Encryption for incoming and outgoing faxes
  • Incoming faxes can be emailed to recipient as PDF, delivered to secure print queue for release, or routed to network directory
  • Outgoing fax status provided via email
  • Integrates with bizhub MFPs
  • No MFP fax kit needed
  • Integrates with Epic EHR
  • 1000 fax pages per MFP per year included in active maintenance contract
  • Additional pages can be purchased

More about OpenText RightFax


RightFax integrates fax distribution with bizhub MFPs, E H R systems, email, and other business applications.


  • Supports insights into performance trends with RightFax analytics
  • Provides full audit trail of fax history
  • One deployment option is fully on-premises RightFax server
  • Second option for hybrid deployment with cloud connectivity combines RightFax server and RightFax Connect
